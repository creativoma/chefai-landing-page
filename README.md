# ChefAI Landing Page

A modern, animated landing page for ChefAI - an AI-powered cooking platform.

## Overview

This project is an animated landing page built with React and Motion, featuring scroll animations and interactive elements for showcasing AI cooking features.

## Features

- Smooth scroll animations with Motion
- Interactive hover effects
- Desktop-optimized design
- Snap scrolling navigation
- TypeScript support

## Tech Stack

- **React 19.1.0** - UI Library
- **TypeScript 5.6.3** - Type Safety
- **Vite 6.3.5** - Build Tool
- **Tailwind CSS 4.1.7** - Styling
- **Motion 12.15.0** - Animations

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
